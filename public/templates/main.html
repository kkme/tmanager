2.0 주요 변화 사항

일반폰과 알뜰폰 구분

* 알뜰폰의 매장 선택 방법에 대한 좋은 의견이 있다면 보내주세요.

대불 매장 이름 앞에 있던 '!'가 '@'로 바뀌었습니다.

** !는 (검색에서) 정상동작하지 않을 수 있으니 입력하지마세요.

전체 제품 = 재고 + 반품 + 판매 제품

* 앞으로 반품은 반품으로 등록해주세요!

제품 입력 간편하게 ->

1. 모델명 선택, 색상 선택, 입고처 선택, 매장 선택 일련번호 입력

2. 저장하면 일련번호만 지우고 나머지 값은 그대로. <-- Clear를 선택하면, 모델명 색상, 입고처 등의 선택 사항도 초기화.

제품 이동 간편하게 ->

* 알뜰폰의 경우 여러 매장으로 보내므로 제품 이동 메뉴를 따로 넣음(재고 - 제품 다건 이동)

                a. 첫번째 매장엔 현재 재고가 있는 매장 선택
                b. 두번째는 입고될 매장 선택
                c. (출고될 매장의) 목록에서 항목을 더블클릭하면 오른쪽으로 이동
                d. 저장 누르면 완료
